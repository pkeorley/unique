API: http://www.pkeorley.ml/api/v1

GET /shortlink/create

Description: Use this method to create a new shortlink that can be clicked to.
To use this method, you need an api key, the name of the shortlink, and a link to which the shortlink will redirect

Example: http://www.pkeorley.ml/api/v1/shortlink/create?key=google&url=https://google.com/&api_key=(api_key)

GET /shortlink/get

Description: Use this method to get your IP key statistics (if you don't have it, ask the site developer for it).
You need an api key to use this method

Example: http://www.pkeorley.ml/api/v1/shortlink/get?api_key=(api_key)

POST /shortlink/create

Description: Use this method to create a new shortlink that can be clicked to.
To use this method, you need an api key, the name of the shortlink, and a link to which the shortlink will redirect

Example: import requests

json = {"key": "google", "url": "https://google.com/"}
headers = {"Authorization": "pLQNGMyCclqOOEUD"}
response = requests.post("http://www.pkeorley.ml/api/v1/shortlink/create", json=json, headers=headers)

print(response.json())

GET /shortlink/delete

Description: Use this method to remove shortlink from database

Example: http://www.pkeorley.ml/api/v1/shortlink/delete?api_key=(api_key)&key=google
